# CAD Files

The final report describes a two-hemisphere 6-inch spherical PETG enclosure with:

- 0.5-inch wall thickness
- Equatorial acrylic LiDAR window slot
- Hinge on one side
- Latch/clip on the opposite side
- Internal cradle for the ESP32, battery, and wiring
- Lower ballast cavity for self-righting behavior

The actual STL/STEP files were not embedded as separate CAD files in the uploaded PDF, so this folder is prepared as a placeholder.
